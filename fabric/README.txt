Various notebooks and code snippets to help you get started on running Ray on FABRIC.
Get started by looking at the readable notebook

    ray_A40_2nodes.html

Please be aware that the FABRIC GPU nodes are in high-demand.
First ensure that your notebook can successfully build a single node before reserving a cluster.
Be aware that software installation can take 1 hour or more.

Consider learning with the pre-built Ray image on CloudLab before tackling a FABRIC experiment.

In general, do not specify num_cpus and num_gpus when running Ray on a cluster.
